# 🐸 Nakudama Name Generator

A name generator for the **Nakudama** — a D&D race of frog-folk — built with React + Vite.

Read more about the language and naming system on the [about page](https://hkly.io/nakudama-name-generator/#/about).

## Features

- Generate Nakudama names across different life stages: **Child, Modern, Formal, Ancient**
- Names built from a small phonology system inspired by frog sounds and reduplication
- Pattern-based generator (repeating names, smooth names, croaky cluster names, and **-naku** formal names)
- About page with lore and a description of this interpretation of the **Naku Naku** language

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
